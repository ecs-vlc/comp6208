COMP2208 Advanced Machine Learing

Course Description

This course provides an introduction to the mathematics of machine learning. We explore topics such as calculus for ML, vector spaces, norms, inner products, eigen decomposition, SVD, symmetry, probabilistic modelling, entropy, etc. We use these to understand generalisation, ensemble methods, SVMs, Wasserstein GANs, Gaussian Processes, minimum description length, etc.
